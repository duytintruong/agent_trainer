# Quickstart

## Prerequisites
- Node.js 18+
- AWS Credentials with access to Bedrock Nova Sonic

## Setup

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Environment Variables**
   Create a `.env.local` file with:
   ```bash
   AWS_ACCESS_KEY_ID=your_key
   AWS_SECRET_ACCESS_KEY=your_secret
   AWS_REGION=us-east-1
   ```

3. **Run Development Server**
   ```bash
   npm run dev
   ```

4. **Access App**
   Open http://localhost:3000 to view the Voice Conversation UI.
